Vectruns R snippets.

This is a personal collection of miscelanous R functions. Each set of functions is fully functional and provided with examples.

These functions make my life with R a little easier. They work for me and it's great if the also work for you. If not, too bad. You got the code so enjoy hacking them to your own needs.

There is no manual. The source is well (*cough*) documented.
